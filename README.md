# LLM Learning Projects

This repository contains a collection of beginner-friendly AI and LLM projects built with Python.

## Projects included

- Project 1: Chat
- Project 1 V2: Chat loop
- Project 2: Weather Tool
- Project 3: Mini RAG
- Project 4: PDF Chatbot
- Web App for Document Generation

## Structure

- AI-Projects/ contains all project folders and notes
- Each project folder has its own README, requirements, and Python code

## Getting started

1. Open the project folder you want to run.
2. Create a virtual environment if needed.
3. Install dependencies using requirements.txt.
4. Add your API key locally in a .env file or environment variable.

## Notes

- Local environment files such as .env are ignored by Git.
- Keep secrets out of the repository.
